# Tealium

Tealium is a customer data platform (CDP) providing REST APIs for iQ Tag Management and AudienceStream — enabling enterprises to manage tags, audiences, attributes, connectors, and real-time event data collection across web, mobile, and server-side channels.

## APIs

- **Tealium Collect HTTP API** — Send authenticated events via HTTP into the Customer Data Hub (100 events/sec limit)
- **Tealium iQ Profiles API** — Programmatic management of iQ Tag Management configuration (tags, load rules, events, variables, extensions)
- **Tealium Visitor Profile API** — Query full visitor profile records from AudienceStream
- **Tealium Visitor Privacy API** — GDPR/CCPA compliance: retrieve and delete visitor records (50 req/sec limit)
- **Tealium Moments API** — High-performance real-time visitor context for personalization and AI (200 req/sec per profile)
- **Tealium Data Connect API** — Bulk import/export of event and audience data (500 events/sec limit)
